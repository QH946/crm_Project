# CRM（客户关系管理系统）

## 介绍

基于SSM的CRM项目

开发环境：IDEA

JDK版本：11.0.15

项目管理工具：Maven3.8.4

Web服务器：Tomcat9.0.59

数据库管理系统：Mysql8.0.28

涉及到的技术：

Spring，SpringMVC，Mybatis，JSP，jQuery，ajax，bootstrap等

实现功能：
1.市场活动
2.线索（潜在客户）
3.联系人
4.交易（商机）
5.统计图表
